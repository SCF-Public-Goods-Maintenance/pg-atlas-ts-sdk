# PG Atlas SDK (@pg-atlas/sdk)

[![NPM Version](https://img.shields.io/npm/v/@pg-atlas/sdk)](https://www.npmjs.com/package/@pg-atlas/sdk)
[![License](https://img.shields.io/badge/license-MIT-blue)](./LICENSE)

The official **PG Atlas SDK** for interacting with the PG Atlas API. Built for developers building on the Stellar and Soroban ecosystem within the Public Goods Atlas framework.

## 🚀 Installation

```bash
npm install @pg-atlas/sdk
# or
pnpm add @pg-atlas/sdk
# or
yarn add @pg-atlas/sdk
```
